# Sistema de Facturación para Farmacia

Proyecto final del curso SC-403 — Desarrollo de Aplicaciones Web y Patrones.

## Descripción

El proyecto consiste en un sistema web para una farmacia que permitirá administrar clientes, medicamentos, proveedores, inventario, usuarios y facturas.

El objetivo es facilitar el control de las ventas, mantener actualizado el inventario y reducir errores en los procesos de facturación.
